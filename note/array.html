• 在 Go 语言中数组是固定长度的数据类型，它包含相同类型的连续的元素
  • 数据类型可以是 int string bool struct
  • 表达形式 var array [5]int array = [5] int {1, 2, 3, 4, 5}
    • 指定位置初始化
    • array := [5]int { 1:77 }
    • 指针数组
      • array := [5] *int{}
    • 二维数组
  • 中括号中必须有固定数字，利用{}直接赋值
  • 数组作为参数传递 
             func foo(arry *[1e6]int) {
            ...
            }

         func main() {
         arry := [1e6]int
         foo(&arry)
         ...
         }
