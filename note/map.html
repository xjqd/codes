• map 是一种无序的键值对序列
  • 表达形式
    • key includes origin type and struct type. not include slice, function,
    • values is the expression
    • map[string]int
    • 字面赋值
      • dict := map[string]string{"color1":"red","color2":"blue"}
    • make 赋值
      • dict := make(map[string]int){"red":1,"blue":2}
      • dict1 := make(map[string]int){} dict1["red"]=2 dict1["blue"]=3
  • 如果不初始化一个map,则会创建一个nil map
    • var map[string]int nil map不能存放键值对
  • 从map里检索一个值 -case1 
               value, exists := colors["blue"]
           if exists {
              fmt.Println(value)
              }
    -case2 
              value := colors["blue"]
           if value != "" {
                fmt.Println(value)
                }
    • 迭代map ( 与取一个键值对不同) 
               colors := map[string]string {
              "AliceBlue":   "#f0f8ff",
              "Coral":       "#ff7F50",
              "DarkGray":    "#a9a9a9",
              "ForestGreen": "#228b22",
              }
         for key, value :=range colors {
            fmt.Println("key=", key, "value=",value)
            }
      *从map中删除键值对 
                delete colors["Coral"]
          colors["Coral"] = "#aaaaa"
    • 在函数间传递map,传递的是指向 
               func remove_ele(colors map[string]string){
